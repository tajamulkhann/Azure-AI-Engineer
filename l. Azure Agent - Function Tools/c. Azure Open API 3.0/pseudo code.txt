   using the Yahoo Finance connection ID.

   - Name: getYahooFinanceData
   - Description: Fetch stock and financial market data
   - Auth: Yahoo Finance connection details

7. Create an AI Agent using GPT-4 model 
   that uses this OpenAPI tool for real-time finance insights.

8. Create a chat thread for interaction.

9. Start a loop to chat with the agent:
   - Take user input (e.g., “Show Apple stock price”)
   - Send the message to the agent
   - Process the run and wait for a response
   - Print the agent’s answer
